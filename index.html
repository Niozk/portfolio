Over mij

Ik ben een developer en studeer daarnaast momenteel aan de Hogeschool van Leiden. Mijn specialisme is programmeren en met name webdevelopment. Programeren heb ik grotendeels in de praktijk geleerd, wat begon als een hobby in 2020 is nu uitgegroeid tot mijn passie.

Naast mijn uitgebreide kennis van diverse technologieën, waaronder Javascript, Vue en Python, ben ik zeer leergierig als het gaat om het vergroten van mijn programmeerervaring en het verwerven van nieuwe kennis binnen dit vakgebied. Bovendien kan ik met volle overtuiging zeggen dat ik een hardwerkend persoon ben met een sterke focus op het behalen van gestelde doelen.

Weather App

Een simpele maar nette weer app waarmee je gemakkelijk het weer kan opzoeken van welke locatie u wilt, via de OpenWeather Api. Deze website heb ik gemaakt met Vue en vanilla HTML, CSS en Javascript. Daarnaast heb ik Axios gebruikt voor de api requests en Motion.js voor de animaties.

Dodo App

Een backend project waarbij de focus was om een todo app te creëren met een volledige account creatie en login systeem. Bij de todo's kun je CRUD operations gebruiken en ze worden ogeslagen per account. Gebruik de browser console om meldingen te zien. Deze applicatie is gemaakt met Vue, Express, Firebase Auth en JSON-Server.

Een frontend project waarbij de focus was om een hotel site te maken met behulp van de component library: PrimeVue. De site hoort een gevoel te geven als een normale website waarbij je een hotel kan boeken. De site is gemaakt met Vue, de Vue-Router, PrimeVue, Pinia en Motion.js.
