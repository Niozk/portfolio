Dodo App

Een backend project waarbij de focus was om een todo app te creëren met een volledige account creatie en login systeem. Bij de todo's kun je CRUD operations gebruiken en ze worden ogeslagen per account. Gebruik de browser console om meldingen te zien. Deze applicatie is gemaakt met Vue, Express, Firebase Auth en JSON-Server.

Een frontend project waarbij de focus was om een hotel site te maken met behulp van de component library: PrimeVue. De site hoort een gevoel te geven als een normale website waarbij je een hotel kan boeken. De site is gemaakt met Vue, de Vue-Router, PrimeVue, Pinia en Motion.js.
